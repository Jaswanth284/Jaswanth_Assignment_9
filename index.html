About This Website

This website presents visualizations for renewable energy usage, CO₂ emissions, forest area, and power consumption for Norway, Spain, Brazil, India, and Egypt from 2010 to 2024. All charts were created in Google Colab with Altair and exported as images.

Each page shows a different type of visualization: scatter plots use color and size to encode multiple variables, bar charts compare values between countries and years, and trend charts show how indicators change over time.
